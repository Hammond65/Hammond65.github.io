Image Inpainting

LaMa: Resolution-Robust Large Mask Inpainting with Fourier Convolutions

February 28, 2026 · Suvorov et al. (WACV 2022)

The Challenge of Large Mask Inpainting

Image inpainting — the task of filling in missing or masked regions of an image — is a fundamental problem in computer vision. While small masks can often be filled using nearby pixel information, large masks present a much harder challenge. When most of the image is missing, the model must understand the global structure and semantics of the scene to generate coherent content.

Traditional approaches using standard convolutional neural networks (CNNs) struggle with this. CNNs have a limited receptive field — each layer only sees a small local region. Even with many stacked layers, the effective receptive field grows slowly, making it difficult to capture global context needed for large mask completion.

Three Key Contributions

1. Fast Fourier Convolutions (FFC)

The most important contribution is replacing standard convolutions in the bottleneck layers with FFC blocks. Originally proposed by Chi et al. (NeurIPS 2020) for image classification, FFCs are designed to capture both local and global features simultaneously.

Each FFC block has two parallel processing paths:

  • Local Path: Acts like an ordinary 3×3 convolution. Takes information from both local and global paths, sums them, then applies normalization and activation.
  • Global Path: Takes information from the local path through a convolution. For global features, applies a Spectral Transform (FFT-based). The Spectral Transform converts the image from the spatial domain (pixels) to the frequency domain (patterns/waves) using 2D Fast Fourier Transform, applies convolution in frequency space, then converts back with inverse FFT.

Why Frequency Domain Captures Global Features

In the spatial domain, each pixel value is local (represents one location). In the frequency domain, each frequency component is global — it represents information from every position in the spatial domain. This bidirectional relationship is key.

FFC vs. Standard Convolution at Higher Resolutions

Consider a U-Net trained on 12×12 face images. At the bottleneck, features are 3×3, and a 3×3 convolution covers 100% of the feature map. Now increase the resolution to 28×28 — at the bottleneck, features become 7×7, and the same 3×3 convolution now covers only ~18.4%. The model can no longer see the global context.

FFC blocks don't have this problem. Each block maintains a complete view of the entire input at any resolution. Whether the image is 256×256 or 2048×2048, the frequency domain representation captures the entire image. The 1×1 convolution in frequency space still operates on similar frequency patterns, and FFT scales with image dimensions.

Ablation Results

Models trained on 256×256 and tested up to 2048×2048 showed that standard LaMa (regular convolution) performance degrades rapidly at higher resolutions, while LaMa with FFC maintains performance across all resolutions. Furthermore, LaMa-Fourier with 27M parameters outperforms standard LaMa with 45M and 74M parameters — achieving better quality with 40-60% fewer parameters.

2. High Receptive Field Perceptual Loss (ResPL)

Traditional pixel-wise losses like L1/L2 require exact reconstruction. For large masks, the visible context is insufficient — multiple valid outputs exist. The model averages them, producing blurry results.

Why not LPIPS? LPIPS was trained on classification tasks with 64×64 patches. It focuses on textures, not global structure. For higher resolution inputs, its global capturing ability diminishes.

LaMa's solution: use features from a ResNet50-dilated segmentation network trained on ADE20K (diverse scenes with semantic understanding). Dilated convolutions provide a large receptive field, capturing global semantic structure rather than just local textures. The ablation study confirms HRFPL improves both FID and LPIPS metrics over standard LPIPS loss.

3. Large Training Masks

The third contribution is an aggressive large mask training strategy. By training with larger, more challenging masks, the model can't rely on copying nearby pixels — there are none. This forces the network to understand and generate content based on global scene understanding, aligning with the FFC architecture's strength.

Architecture Details

  • Input: Masked image (RGB) + binary mask → 4 channels
  • Initial Conv: 7×7 with 3-pixel padding
  • Encoder: 3 downsampling blocks (8× total downsampling)
  • Bottleneck: 6-18 FFC ResNet blocks with global + local branches
  • Decoder: 3 upsampling blocks (mirror encoder)
  • Training: Adam optimizer, 1M iterations, batch size 30, trained on Places and CelebA-HQ
  • Loss: Lfinal = κLadv + αLHRFPL + βLDiscPL + γR₁

Practical Results

LaMa excels at repetitive structures (fences, patterns) for small-to-medium masks, reconstructing them nearly perfectly. For text on clothing with proper segmentation masks, results are also strong. However, challenges remain:

  • Large masks with complex structures show some distortion
  • Masks covering multiple objects without complete segmentation produce inconsistent results
  • Proper segmentation masking is critical — the model requires clear, complete object boundaries

Conclusion

LaMa demonstrates that Fast Fourier Convolutions can fundamentally change how inpainting networks process information. By capturing both local and global features simultaneously, achieving resolution robustness, and using 40-60% fewer parameters, FFC-based inpainting represents a significant advance. Combined with the high receptive field perceptual loss and aggressive masking strategy, LaMa sets a strong foundation for practical image restoration applications.
